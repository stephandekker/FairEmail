# Permanent Delete from Conversation

## Parent Feature
#3.2 Conversation Threading

## User Story
As any user, I want the ability to permanently delete a message from a conversation, removing it from the thread entirely, so that I can truly remove unwanted messages.

## Blocked by
6-conversation-view

## Acceptance Criteria
- [ ] The user can permanently delete a message from within the conversation view (FR-38).
- [ ] Permanently deleting a message removes it from the thread; the remaining messages stay grouped (AC-15).
- [ ] The conversation's message count updates to reflect the deletion.

## HITL / AFK
AFK — a single destructive action with clear semantics.
