# Horizontal Rule Insertion

## Parent Feature
#4.1 Rich Text Editor

## User Story
As a visual communicator, I want to insert a horizontal rule (visual line separator) at the cursor position, so that I can divide my message into distinct sections.

## Blocked by
`2-formatting-toolbar-shell`

## Acceptance Criteria
- A toolbar button for inserting a horizontal rule is present and functional.
- Tapping the button inserts a visible horizontal line at the current cursor position.
- The line spans the width of the message body.
- The horizontal rule appears in the sent HTML as a recognized separator element.
- The rule persists after saving as a draft and reopening.
