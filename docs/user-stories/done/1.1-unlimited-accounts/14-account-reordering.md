# Account Reordering

## Parent Feature
#1.1 Unlimited Accounts

## User Story
As any user, I want to reorder accounts via drag-and-drop, so that I can arrange them in the order that makes sense to me, and I want to be able to reset to the default order.

## Blocked by
1-create-imap-account

## Acceptance Criteria
- Accounts in the account list / navigation pane can be reordered via drag-and-drop (FR-20, US-19).
- The custom order persists across application restarts (FR-20, AC-8).
- A "reset order" action restores the default sort: primary first, then alphabetical (FR-21, US-20).
- Drag-and-drop is keyboard-accessible (NFR-7).
