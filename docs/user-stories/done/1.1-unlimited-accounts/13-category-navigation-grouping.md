# Category Grouping in Navigation Pane

## Parent Feature
#1.1 Unlimited Accounts

## User Story
As a multi-account user, I want accounts to be grouped under their category headers in the navigation pane when category display is enabled, so that I can visually locate accounts by their organizational group.

## Blocked by
12-account-category-labels

## Acceptance Criteria
- When category display is enabled, accounts with category labels are grouped under category headers in the navigation pane (FR-18, AC-7).
- Categories are sorted alphabetically; within each category, accounts are sorted by custom order (FR-18).
- Accounts without a category appear in an uncategorized group.
- Category display can be toggled on/off via a setting (FR-21 variant from US-21).
- When category display is disabled, accounts are sorted by custom order, then primary first, then alphabetically (FR-19).
