## Parent Feature

#4.14 Attachments

## What to build

Provide a mechanism in the compose window to attach a vCard (contact card). The resulting attachment is a .vcf file with the correct MIME type (text/vcard).

Covers epic sections: US-11, FR-7, AC-6.

## Acceptance criteria

- [ ] The compose window offers an action to attach a vCard.
- [ ] The attached vCard is a .vcf file with MIME type text/vcard (AC-6).
- [ ] The vCard contains valid contact information in standard vCard format.
