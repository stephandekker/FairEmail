## Parent Feature

#4.14 Attachments

## What to build

Allow users to rename an attachment in the compose attachment list before sending. The rename updates both the displayed name and the filename that will be sent to recipients.

Covers epic sections: US-22, FR-25, AC-11.

## Acceptance criteria

- [ ] Each attachment in the compose list offers a rename action.
- [ ] Renaming updates the displayed filename in the list (AC-11).
- [ ] The renamed filename is the one sent to recipients (AC-11).
- [ ] Renaming works offline.

## Blocked by

- Blocked by `2-compose-attachment-list`
