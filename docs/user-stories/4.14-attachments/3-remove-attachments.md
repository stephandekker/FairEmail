## Parent Feature

#4.14 Attachments

## What to build

Enable removal of attachments from the compose draft — both individually and in bulk. Each item in the attachment list offers a remove action. A separate "remove all attachments" action clears the entire list in one step.

Covers epic sections: US-20, US-21, FR-23, FR-24, AC-10.

## Acceptance criteria

- [ ] Each attachment in the list has a remove action that removes only that item (AC-10).
- [ ] A "remove all attachments" action clears the entire attachment list (AC-10).
- [ ] Removal works offline (NFR-6).
