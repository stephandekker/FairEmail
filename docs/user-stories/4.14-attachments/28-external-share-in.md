## Parent Feature

#4.14 Attachments

## What to build

Accept files shared from other applications via the system share mechanism. Shared files appear as attachments in a new or existing draft. A setting controls whether shared files create a new draft or are appended to the currently open draft.

Covers epic sections: US-3, FR-6, FR-48.

## Acceptance criteria

- [ ] Files shared from another application into the email client appear as attachments in a draft.
- [ ] A setting controls whether shared files create a new draft or append to the currently open draft (FR-48).
- [ ] Shared files are detected with correct MIME type and sanitized filename.
